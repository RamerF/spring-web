# spring-web
A demo web project use Spring Boot deploy.

### Demo list

- Spring Boot
- Spring-Data-Jpa CRUD
- Spring-Security
- Thymeleaf
- JQuery tools
  - $.alert 
  - $.confirm
  - $(selector).ajaxTable
  - ...
 - Jdbc CRUD
 - Using OkHttpClient to process http request
 - Convert JSON / XML to object and inverse
 - ...
